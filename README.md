The working compiled program is atlas-packer in the dist folder. The code for it is the atlas_packer.py and build is just build.sh. Was created on AntiX os.

HOW TO USE
Select your images and drag and drop into the GUI. Any image file should work. If you want to add more textures to a texture atlas that you already have, import the images that you want to add first then import your texture atlas to create a new one. Empty space should be filled first.
